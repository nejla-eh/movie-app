Movie API

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

List all movies

GET /movies none

A JSON object holding data about all the movies 

[{
    ID: Integer,
    Title: String,
    Description: String,
    Genre: String,
    Director: String,
    ImagePath: String,
    Featured: Boolean
}]

Get a movie

GET /movies/[Title] none

The Movie title in the URL

Add a movie

POST /movies

A JSON object holding data about the movie to be added 

{
    Title: String,
    Description: String,
    Genre: String,
    Director: String,
    ImagePath: String
}

A JSON object holding data about an added movie 

{
    ID: Integer,
    Title: String,
    Description: String,
    Genre: String,
    Director: String,
    ImagePath: String,
    Featured: Boolean
}

Remove a movie

DELETE /movies/[Title]

The Movie title in the URL

A message indicating whether the movie was successfully removed 

{
    ID: Integer,
    Title: String,
    Description: String,
    Genre: String,
    Director: String,
    ImagePath: String,
    Featured: Boolean
}

Get a genre

GET /genres/[Genre]

The Genre name in the URL

A JSON object holding data about a genre 

{
    Name: String,
    Description: String,
}

Get a director

GET /directors/[Director]

The Directors name in the URL

A JSON object holding data about a director 

{
    Name: String,
    Bio: String,
}

List all users

GET /users none

A JSON object holding data about all users 

[{
    Username: String,
    Password: String,
    Email: String,
    Birthday: Date,
    FavouriteMovies: [Movie]
}]

Create a new user

POST /users

A JSON object holding data about the new user: 

{
    Username: String,
    Password: String,
    Email: String,
    Birthday: Date,
    FavouriteMovies: [Movie]
}

A JSON object holding data about a registered user 

{
    ID: Integer,
    Username: String,
    Password: String,
    Email: String,
    Birthday: Date,
    FavouriteMovies: [Movie]
}

Update an user

PUT /users/[Username]

The Username in the URL

A JSON object holding data of the updated user 

{
    Username: String,
    Password: String,
    Email: String,
    Birthday: Date,
    FavouriteMovies: [Movie],
}

Delete an user

DELETE /users/[Username]

The Username in the URL

A message indicating whether the user was successfully deregistered

Add a movie to the user's list of favorite movies

POST /users/[Username]/movies/[MovieID]

The Username and MovieID in the URL

A JSON object holding data of the updated user 

{
    Username: String,
    Password: String,
    Email: String,
    Birthday: Date,
    FavouriteMovies: [Movie],
}
